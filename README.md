**Práctica – Ejercicio 3: Encaminador IP (Filtrado de paquetes mediante ACLs)**

En esta práctica se configura un encaminador IP aplicando listas de control de acceso (ACLs) para filtrar el tráfico de red. 
Se definen reglas que permiten o deniegan paquetes en función de criterios como dirección IP, protocolo o puerto, analizando 
cómo estas políticas mejoran la seguridad y el control del tráfico en una red.


**Simulacro de examen práctico – Seguridad en Sistemas Informáticos**

Simulacro de examen práctico – Seguridad en Sistemas Informáticos
Ejercicio práctico global centrado en el diseño e implementación de una política de seguridad perimetral en una red empresarial.
Se configuran reglas de filtrado mediante ACLs o firewall para controlar el tráfico entrante y saliente (HTTP/HTTPS, DNS, correo, ICMP),
se restringe el acceso a servicios internos y públicos, y se aplican medidas de protección en los dispositivos de red. 

El objetivo esgarantizar la confidencialidad, integridad y control de acceso según los requisitos establecidos.
